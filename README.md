# Stratagem Hero

A python (pygame) recreation of the Stratagem Hero game included with the Super Citizen Edition of Helldivers 2.

## Updates

- Open to suggestions and improvements to both code and game logic
- Selectable Gamemodes
- Online Leaderboard
